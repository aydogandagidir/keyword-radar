# Task 05 — Floating Panel UI

Build the extension panel UI.

Required UI:
- Marketplace label.
- Seed keyword input.
- Expansion options.
- Collect button.
- Progress indicator.
- Stop/cancel button if feasible.
- Keyword results table.
- Word frequency section.
- Marketplace comparison section.
- Copy button.
- CSV export button.
- XLSX export button.
- Error and empty states.

Acceptance criteria:
- User can enter seed keyword.
- User can start collection.
- Results appear in the table.
- UI remains responsive.
- Empty state is clear.
