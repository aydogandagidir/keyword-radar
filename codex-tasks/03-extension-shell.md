# Task 03 — Chrome Extension Shell

Create the Manifest V3 Chrome Extension in `apps/extension`.

Requirements:
- Vite + React + TypeScript.
- Valid `manifest.json`.
- Background service worker.
- Content script.
- Floating panel injection.
- Marketplace detection via adapter router.
- Minimal permissions.
- Host permissions for Amazon.com.tr, Trendyol, Hepsiburada, n11.

Acceptance criteria:
- Extension builds.
- Extension can be loaded unpacked.
- Panel appears on supported marketplaces.
- Panel does not appear on unsupported pages.
