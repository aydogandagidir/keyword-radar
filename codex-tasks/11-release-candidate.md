# Task 11 — Release Candidate

Prepare MVP release candidate.

Run:
- pnpm install
- pnpm build
- pnpm test

Then:
- Fix critical issues.
- Verify extension build output.
- Verify docs.
- Verify privacy and permissions.
- Produce release notes.

Acceptance criteria:
- Build passes.
- Tests pass.
- Manual QA checklist can be executed.
- Release notes generated.
