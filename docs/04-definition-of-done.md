# Definition of Done

## Project-level DoD

A task is done only when:

- Code is implemented.
- TypeScript passes.
- Tests are added or updated.
- Relevant docs are updated.
- No private data collection is introduced.
- Build passes.
- Acceptance criteria are met.

## MVP release DoD

- `pnpm install` works.
- `pnpm build` works.
- `pnpm test` works.
- Chrome Extension loads unpacked.
- Supported marketplace domains are detected.
- Floating panel appears only where expected.
- User can enter seed keyword.
- User can collect keyword suggestions.
- Suggestions are deduped and normalized.
- CSV export works.
- XLSX export works.
- Word frequency analysis works.
- Marketplace comparison works.
- Backend skeleton starts.
- Privacy documentation exists.
- Permissions documentation exists.
- Known limitations are documented.
