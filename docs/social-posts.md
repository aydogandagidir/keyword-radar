### EN (uzun form)

```
A new Chrome extension for Turkish marketplace sellers is now live on the Chrome Web Store: Keyword Radar. 🛰️

If you sell on Amazon.com.tr, Trendyol, Hepsiburada or n11, you know the silent problem: which words are shoppers actually using to find your product? Writing titles and tags from guesswork means invisibility — and most search-volume tools either don't cover Turkish marketplaces or cost a fortune.

At Bluedev, we built a privacy-first extension that turns each marketplace's own autocomplete suggestions — what shoppers really type — into a structured, scored keyword list you can act on:

🔍 Expands one seed keyword into dozens of probes; collects the visible suggestions across all 4 marketplaces
📊 Opportunity score (0–100) — frequency + long-tail + marketplace coverage + confidence in a single number
🧩 Listing Gap analysis — shows the high-value keywords missing from your title and description
📤 One-click CSV + a multi-sheet XLSX report (scores, word frequency, coverage, gaps) — charted, ready to ship
🔒 No account, no login, no telemetry — everything stays local in your browser. It only reads the autocomplete text a shopper already sees; never seller-panel, order or customer data

Free and live now 👇
```
