# Privacy Policy Draft

Bluedev Marketplace Keyword Radar helps users collect and analyze marketplace autocomplete keyword suggestions.

## Data processed

The extension may process:

- seed keywords entered by the user
- autocomplete keyword suggestions visible in the user's browser
- marketplace name
- keyword run metadata
- optional saved keyword projects

## Data not collected

The extension does not collect:

- marketplace passwords
- seller account credentials
- customer data
- order data
- payment data
- private seller dashboard information

## Storage

Keyword results may be stored locally in the browser or optionally sent to the user's own backend workspace if the user chooses to save a project.

## Third parties

MVP does not send keyword data to paid AI providers by default. Future AI features must require clear configuration and user consent.

## User control

Users can export, copy, or delete their keyword results.
